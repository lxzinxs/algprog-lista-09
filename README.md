# 🧩 POO: Modelagem de Objetos do Mundo Real

Este repositório contém a atividade prática de introdução à **Programação Orientada a Objetos (POO)**. O objetivo principal do exercício foi exercitar a abstração mental, mapeando entidades do cotidiano para os conceitos estruturais de **Atributos** (características/dados) e **Métodos** (ações/comportamentos).

---

## 💡 Conceitos Aplicados

A modelagem seguiu os pilares iniciais da estrutura de classes em linguagens de programação:
* **Classe**: O molde ou abstração de uma entidade (ex: `Carro`, `Pessoa`).
* **Atributos (Variáveis de Instância)**: Estado ou propriedades que definem o objeto.
* **Métodos (Funções)**: O comportamento que o objeto pode executar ou as mensagens às quais ele responde.

## 🚀 Relevância para o Desenvolvimento de Software

Mapear o mundo real dessa forma é o passo fundamental antes de abrir a IDE e digitar códigos em Java ou C#. Compreender a separação entre o que um objeto **é** (dados) e o que um objeto **faz** (funções) facilita a construção de códigos limpos, modulares e reutilizáveis.
